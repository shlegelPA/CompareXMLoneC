Тестовая версия (сыроежка)


Обработка предназначена для сравнения XML файлов Метаданных конфигурации, которые находятся в конфликте слияния (MERGE_Conflict).


Основной принцип:

  1. Поиск конфликтных файлов в репозитории GIT (выбрать каталог, содержащий директорию .git)

  2. Получение трех версий файла из репозитория. Версия текущей ветки, вливаемой ветки и версия общего предка. (Двойной клик по строке с файлами)

  3. Преобразование XML в дерево формы, через XDTO. (Автоматически, при выборе файла из табличной части)

  4. В дереве выводится три версии, у каждой версии показано количество различий, строки с различиями выделены розовым (развернуть дерево, проверить)

  5. При загрузке флажки версии устанавливаются автоматически <Алгоритм дорабатывается, ручная проверка обязательна и не только> (установка флажков в дереве)

  6. Сборка результата, с помощью XDTO (собрать Исходник)


Адрес проекта https://github.com/shlegelPA/CompareXMLoneC
